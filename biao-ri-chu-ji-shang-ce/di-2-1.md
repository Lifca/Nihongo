# 第１７課

### 课文

书面：

1. わたしは新しい洋服**が欲しいです**。
2. わたしが映画を**見たいです**。
3. 一緒にお茶を**飲みませんか**。
4. ちょっと**休みましょう**。

对话：

1. ー今何が一番欲しいですか。

   ー安い車**が欲しいです**。

2. ー今日デパートへ買い物に行きます。李さんも一緒にどうですか。

   ーはい，ぜひ**行きたいです**。

3. ー李さん，何を**食べたいですか**。

   ー**何でも**いいです。

4. ー土曜日の午後，コンサートを**行きませんか**。

   ーいいですね。**行きましょう**。

### 语法

#### ＜名＞が欲しいです

表示愿望。前可加“＜名＞＋は～”。

#### ＜名＞を＜动＞たいです

想~。前可加“＜名＞＋は～”。

{% hint style="info" %}
动词为ます形的词干。
{% endhint %}

#### ＜动＞ませんか

表示提议。

#### ＜动＞ましょう

表示提议。

#### 疑问词＋でも

表示在任何情况下事态都相同。全面肯定。
