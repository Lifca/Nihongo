# 第１９課（ない形）

### 课文

书面：

1. 部屋の鍵を**忘れないでください**。
2. 李さんは今日早く**帰らなければなりません**。
3. 明日は**残業しなくてもいいですよ**。
4. あの人**が**吉田課長ですよ。

对话：

1. ーその品物に**触らないでください**。

   ーあっ，すみません。

2. ー李さん，一緒に食事に行きませんか。

   ーすみません。今日は早く帰って，レポートを**書かなければなりません**。

3. ー先生，もう薬を**飲まなくてもいいですか**。

   ーはい，いいですよ。

4. ー吉田課長はいますか。

   ーわたし**が**吉田ですが...。

### 语法

#### ＜动ない＞でください

 请不要~。表示否定的命令。

#### ＜动＞なければなりません

表示必须。

{% hint style="info" %}
变化：～ない　⇒　～なければ。
{% endhint %}

#### ＜动＞なくてもいいです

表示不做某事也可以。相当于上一句的否定。

{% hint style="info" %}
变化：～ない　⇒　～なくても。
{% endhint %}

#### ＜名１＞が＜名２＞です

名词1是名词2。名词1为新信息，名词2为旧信息。

{% hint style="info" %}
何、だれ、どこ、いつ等做主语只能用が。
{% endhint %}
