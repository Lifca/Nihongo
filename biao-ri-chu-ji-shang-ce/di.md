# 第１課

### 课文 {#ke-wen}

书面：

1. 李さん**は**中国人**です**。
2. 森さん**は**学生**ではありません**。
3. 林さんは日本人です。
4. 李さんはJC企画**の**社員です。

对话：

1. ーわたしは李です。小野さん**ですか**。

   ー**はい，そうです**。小野です。

2. ー森さんは学生ですか。

   ーいいえ，学生ではありません。

3. ー吉田さんですか。

   ー**いいえ，ちがいます**。森です。

4. ー李さんはJC 企画の社員ですか。

   ーはい、そうです。

### 语法 {#yu-fa}

#### ＜名＞は＜名＞です {#hadesu}

~是~。は提示主题。

#### ＜名＞**は＜名＞ではありません** {#hadehaarimasen}

~不是~。口语中では→じゃ。

#### ＜名＞**は＜名＞ですか** {#hadesuka}

~是~吗？

{% hint style="info" %}
回答：

* はい，そうです。
* いいえ，違います。
{% endhint %}

#### ＜名＞**の＜名＞** {#no}

前者是后者从属的机构、国家或属性。
